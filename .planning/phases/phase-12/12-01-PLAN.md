**Step 2: GREEN — Implement pollSessions**

Implement `pollSessions` in `src/polling.ts` with this algorithm:

```
1. const interval = options?.interval ?? 10000
2. const timeout = options?.timeout ?? 600000
3. const failFast = options?.failFast ?? false
4. const start = Date.now()
5. const completed = new Set<string>(), failed = new Set<string>(), cancelled = new Set<string>()
6. Helper markTerminal(id, status): adds to correct Set, calls callbacks?.onTerminal(id, status)
7. While Date.now() - start < timeout AND NOT (failFast AND failed.size > 0):
   a. remaining = sessionIds.filter(id => !completed.has(id) && !failed.has(id) && !cancelled.has(id))
   b. If remaining.length === 0 → break
   c. For each id in remaining:
      - try: session = client.getSession(id), { activities } = client.listActivities(id, 10), status = deriveStatus(session, activities)
      - if completed → markTerminal(id, 'completed')
      - if failed → markTerminal(id, 'failed')
      - if cancelled → markTerminal(id, 'cancelled')
      - catch: /* transient — continue */
   d. callbacks?.onPoll({ completed: completed.size, failed: failed.size, cancelled: cancelled.size, remaining: remaining.length })
   e. If failFast AND failed.size > 0 → break
   f. Re-check remaining; if 0 → break
   g. await sleep(interval)  // private sleep function in polling.ts
8. stillRunning = sessionIds.filter(id => !completed.has(id) && !failed.has(id) && !cancelled.has(id))
9. return { completed: [...completed], failed: [...failed], cancelled: [...cancelled], stillRunning, timedOut: stillRunning.length > 0 }
```

Import `deriveStatus` from `./client.js`.

Add private `sleep` helper: `function sleep(ms: number): Promise<void> { return new Promise(r => setTimeout(r, ms)); }`

   ```typescript
   export async function waitForCompletion(
     client: JulesClient,
     _config: JulesConfig,
     sessionIds: string[],
     options: WaitOptions = {},
   ): Promise<PollResult> {
     info(chalk.bold(`\nWaiting for ${sessionIds.length} session(s) to complete (timeout: ${(options.timeout ?? 600000) / 1000}s)...\n`));

     const callbacks: PollCallbacks = {
       onPoll: ({ failed, remaining }) => {
         if (!isJson()) {
           console.log(
             chalk.dim(`  ${remaining} session(s) still running`) +
             (failed > 0 ? chalk.red(` [${failed} failed]`) : ''),
           );
         }
       },
     };

     const result = await pollSessions(client, sessionIds, {
       interval: options.interval ?? 30000,
       timeout: options.timeout ?? 600000,
       failFast: options.failFast,
     }, callbacks);

     emit(
       () => {
         if (!result.timedOut) console.log(chalk.green.bold(`✓ All sessions resolved.`));
         else console.log(chalk.yellow(`Timeout reached. ${result.stillRunning.length} session(s) may still be running.`));
         console.log(
           `  ${chalk.green(`completed: ${result.completed.length}`)}` +
           (result.failed.length > 0 ? `, ${chalk.red(`failed: ${result.failed.length}`)}` : '') +
           (result.cancelled.length > 0 ? `, ${chalk.dim(`cancelled: ${result.cancelled.length}`)}` : ''),
         );
       },
       { event: 'final', ...result },
     );

     return result;
   }
   ```
