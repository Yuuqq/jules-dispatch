    **New export — `createMcpServer(config, client)`:**

    ```typescript
    export function createMcpServer(config: JulesConfig, client: JulesClient): McpServer {
    ```

    This function contains ALL existing logic from `runMcpServer()` EXCEPT:
    - The `loadConfig()` call (config is passed in)
    - The `new JulesClient(config)` call (client is passed in)
    - The `await server.connect(new StdioServerTransport())` call at the end

    It returns the configured `server` instance.

    **Refactored `runMcpServer()`:**

    ```typescript
    export async function runMcpServer(options: McpServerOptions): Promise<void> {
      const config = loadConfig(options.projectDir, {
        apiKeyOverride: options.apiKeyOverride,
        noExit: true,
      });
      const client = new JulesClient(config);
      const server = createMcpServer(config, client);
      await server.connect(new StdioServerTransport());
    }
    ```

    **New imports needed in mcp.ts:**

    Add `import type { JulesConfig } from './types.js';` if not already present (check current imports — `JulesConfig` may need to be imported since it's now a parameter type).
